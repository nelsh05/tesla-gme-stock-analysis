# tesla-gme-stock-analysis
Data extraction and analysis project using Python, Pandas and BeautifulSoup

# Tesla & GameStop Stock Analysis
This project was completed as part of the IBM Data Analyst course.

## Project Overview
- Extracted stock data using yfinance
- Scraped revenue data using BeautifulSoup
- Cleaned and transformed data using pandas
- Visualized stock price and revenue using matplotlib

## Tools Used
- Python
- Pandas
- BeautifulSoup
- Matplotlib
- yfinance
